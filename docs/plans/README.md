# docs/plans/

Implementation plans and project roadmaps for Corporate Finance.

## Purpose

Stores phase-by-phase plans: requirements analysis, model development, testing, and delivery. Each plan includes timelines, ownership, and completion criteria.

## File Structure

```
# <Plan Title>

## Objective
## Scope
## Milestones
## Ownership
## Risks & Mitigation
## Success Criteria
```

## Examples

- `phase1_data_collection_plan.md`
- `phase2_model_build_plan.md`
- `q2_2026_roadmap.md`
